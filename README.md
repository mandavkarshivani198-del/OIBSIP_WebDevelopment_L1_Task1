## Objective
Create a responsive landing page for an online learning platform using HTML and CSS.  
This project helps beginners learn layout design, sectioning, headers, footers, image placement, buttons, and responsive styling.

## Tools Used
- HTML5
- CSS3
- Google Fonts (Poppins & Inter)
- VS Code (Code Editor)
- Chrome / Firefox (Browser)

## Features
- Hero section with heading, description, and call-to-action button
- Courses section with multiple course cards
- Informational sections with images and text
- Features section highlighting platform benefits
- Footer with copyright information
- Fully responsive layout for desktop and mobile screens
- Hover effects for buttons and course cards

## Steps Performed
1. Created `Landing_Page.html` with semantic sections: header, hero, courses, informational sections, features, and footer.  
2. Styled page using `Landing_Page.css` for layout, colors, fonts, buttons, hover effects, and spacing.  
3. Added images in the `images/` folder and linked them properly in HTML.  
4. Tested layout on desktop and mobile devices to ensure responsiveness.  
5. Added interactive hover effects for better user experience.

## Outcome 
- Fully functional and visually appealing landing page
- Proper alignment of all sections without overlapping
- Responsive design working on all screen sizes
- Buttons and course cards have smooth hover effects
- Demonstrates basic web development skills suitable for beginners
